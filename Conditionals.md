# Conditional statments

## if-else

```js
if (condition) {
  // statements
} else {
  // Statements
}
```

- Add `else` only if you want to do something else when the condition is false.

## switch

```js
switch (expression) {
  case value1:
    // statements
    break;
  case value2:
    // statements
    break;
  case valueN:
    // statements
    break;
  default:
  // statements
}
```
